# HOW TO GET STARTED

1. Install node_modules in both server and client (cd into the directories).
2. Clone the .env.example as .env, and add the values to it
3. Setup your DB
4. Build both the server and client code (cd into the directories and use npm run build).
5. Run the server (cd into the directory and use npm run start, alternatively pm2 start dist/index.js with pm2)
6. Open the html file to use

## NOTE

You may have to change the port the server is hosted, and the url for the client to fetch from.
